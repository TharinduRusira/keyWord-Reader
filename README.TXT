how to run the program in command prompt------

1) unzip the folder
2) go to the dist folder
3) feed the input.txt file with the input
4) save input.txt
5) open the dist folder using command prompt
6) make sure you have set your path to \java\jdk\bin
7) now give the command java -jar "projectalgo.jar"
8) you will see the output on the screen if the input format is correct
9) if not, check output.txt in the dist folder for more details.
